JavaScript Comparisons Exercises

Practice these exercises to master comparison operators in JS.

Exercise 1: Basic Numeric Comparisons

Predict the result (true or false) for each:

  1. 5 > 3
  2. 5 < 5
  3. 5 >= 5
  4. 5 <= 4
  5. 5 != 5

Exercise 2: Lexicographical String Comparison

Determine true/false for each string comparison:

  1. "apple" > "banana"
  2. "ABC" < "abc"
  3. "Cat" > "Car"
  4. "2" > "12"
  5. "Zoo" < "Ant"

Exercise 3: Mixed-type Comparisons & Type Coercion

Predict the result:

  1. "5" > 3
  2. "5" == 5
  3. "05" == 5
  4. true == 1
  5. false == 0

Exercise 4: Strict vs Loose Equality

Decide true/false (pay attention to type):

  1. 0 == false
  2. 0 === false
  3. "" == false
  4. "" === false
  5. null == undefined
  6. null === undefined

Exercise 5: Null/Undefined with Comparisons

Predict outcomes:

  1. null > 0
  2. null == 0
  3. null >= 0
  4. undefined > 0
  5. undefined == 0

Exercise 6: Boolean Truthiness in Comparisons

  1. !!"" == false
  2. !!"0" == true
  3. "" == 0
  4. " " == 0

Exercise 7: Real-world Scenario Challenges

  1. Form puzzle: Input = "" from a text field. Which check is safer, == or ===? Why?
  2. Game logic: A health variable is null. Why might `if (health > 0)` fail?
  3. User profile: Age entered via prompt. Should you use `age == 18` or convert first? Explain.
