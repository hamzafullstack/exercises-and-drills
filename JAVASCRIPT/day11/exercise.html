Conditional Branching in JavaScript | Exercises (Solution is in solution.js)

Scope: Based strictly on javascript.info | Conditional branching: if, ternary operator. solutions included in separate file named solution.js.

Key Concepts You Learned

  • Basic if syntax & structure: run a block only when a condition is truthy.
  • Truthiness & Falsiness: values like 0, "", null, undefined, NaN are falsy; most others are truthy.
  • Chaining with if...else if...else: handle multiple distinct cases with a clear fallback.
  • Brace discipline: use curly braces to avoid ambiguous dangling else bugs when nesting.
  • Ternary operator condition ? A : B: compact alternative for simple two-branch decisions.

Exercises (No Solutions)

Exercise 1: Truthy or Falsy?

Task: Prompt the user for a value. If it is truthy, show "Truthy!"; otherwise, show "Falsy!".

Purpose: Practice how JavaScript evaluates values inside if conditions using truthiness rules.

Exercise 2: Nested Conditions & Braces Safety

Task: Given two numbers a and b:

  • If a > b, show "a is bigger". Inside that same block, if a - b > 10, also show "Significantly bigger!".
  • Otherwise, show "a is not bigger".

Purpose: Reinforce nested if usage and the habit of always using braces for clarity and correctness.

Exercise 3: Multi-Way Condition (if...else if...else)

Task: Prompt for a mark (0–100) and show a grade:

  • A for >= 90
  • B for 70–89
  • C for 50–69
  • D for < 50

Purpose: Practice multi-branch decision-making with clearly ordered conditions and a full coverage of ranges.

Exercise 4: Ternary Shortcut

Task: Prompt for a number. Using the ternary operator, decide whether it is even or odd and show "Even" or "Odd".

Purpose: Learn when a concise ternary expression is cleaner than a full if...else for a simple binary choice.

Exercise 5: Mix Blocks with Ternary

Task: Prompt for a number n:

  • If n > 0, show "Positive".
  • Otherwise, show "Negative" or "Zero" using a ternary expression.

Purpose: Balance readability (with blocks) and concision (with a ternary) in one small flow.
