JavaScript Fundamentals — Node.js Drill Instructions

Topics: Hello, world · Code structure · "use strict" · Variables · Data types · Type Conversions

1) Hello, world!

  1. Log "Hello, JavaScript!" to the console.
  2. Log "Hello, JS Warrior!" to the console using a different statement.

2) Code structure

  1. Create a single JS file that prints multiple statements to console.
  2. Add single-line and multi-line comments describing what each statement does.

3) The modern mode — "use strict"

  1. Enable strict mode at the top of your file.
  2. Try assigning a value to an undeclared variable and see the error.
  3. Remove strict mode and compare behavior.

4) Variables

  1. Declare let x = 10; and const y = "fixed";.
  2. Try reassigning y and note the error.
  3. Declare var z = 5; var z = 6; and observe behavior.

5) Data types

  1. Create variables for each primitive type: string, number, boolean, null, undefined, bigint, symbol.
  2. Use console.log(typeof <variable>) to check type.
  3. Compare two symbols with same description; show that they are unique.

6) Type Conversions

  1. Convert string "123" to number using Number(...), unary +, and parseInt(...).
  2. Convert 0, "", null, undefined, " " to Boolean and log results.
  3. Convert true, false, 123, 0, NaN, "" to string and log results.
