JS Type Conversion Flashcards

Q1: Convert "42" (string) to number
A1: Number("42") → 42
Q2: Convert empty string "" to boolean
A2: Boolean("") → false
Q3: Convert null to number
A3: Number(null) → 0
Q4: Convert invalid number string "123abc" to number
A4: Number("123abc") → NaN
Q5: Convert string "0" to boolean
A5: Boolean("0") → true
