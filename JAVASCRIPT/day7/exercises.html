JS Type Conversion Exercises

Rules:
  • Do not copy answers from the internet.
  • Write the code yourself based on your understanding.
  • Use console or alerts to test your code.
  • Try converting between String, Number, and Boolean types.
  • Document your learning by adding comments to each line.

Exercise 1 – String to Number

Create a variable userInput containing a numeric string like "42".

Convert it to a number and display the result using alert.

Exercise 2 – Boolean Conversion

Create a variable emptyInput with an empty string "". Convert it to Boolean and display the result.

Exercise 3 – Null to Number

Create a variable randomValue and assign it null. Convert it to a number and display the result.

Exercise 4 – Invalid Number Conversion

Create a variable numAndStr with a string containing numbers and letters like "123abc". Convert it to a number and display the result.

Exercise 5 – Boolean Conversion of Non-Empty Strings

Create a variable strZero containing string "0". Convert it to Boolean and display the result. Try also with a string containing a single space " ".
