JavaScript Variables — Drill Sheet

Scope: Only the javascript.info/variables section.

1) Declare and Assign

  1. Declare a variable named greeting.
  2. Assign the string "Hello" to it in a separate statement (not combined with declaration).
  3. In one line, declare and assign another variable farewell with value "Goodbye".
  4. Log both with console.log.

2) Reassign Value

  1. Declare let mood = "happy";
  2. Later in the script, reassign it to "excited".
  3. Log the final value of mood.

3) One per Line

Declare the following on separate lines (not a single comma-separated statement):

let name = "Alice";
let age = 30;
let city = "Dubai";

4) Invalid vs Valid Names — Identify Errors

For each line below, write it and note whether it is valid or throws an error. Briefly explain why.

let 1stPlace = "Gold"; 
let my-name = "John"; 
let _status = "active"; 
let $ = 100;

5) Case Sensitivity

  1. Declare let Score = 10;
  2. Declare let score = 20;
  3. Log both to show they’re different identifiers.

6) Reserved Word Check

  1. Attempt let let = 5;
  2. Record the error and explain why it fails.

Bonus (Optional)

7) Copy Value

  1. Declare let original = "First";
  2. Declare let copy; then set copy = original;
  3. Log both to confirm they hold the same value.

8) Re-Declaration Error (Same Block)

  1. Declare let item = "Book";
  2. Attempt to declare let item = "Pen"; again in the same block.
  3. Capture the error message and summarize it.

Why this matters

Nail the fundamentals: valid names, assignment, re-assignment, case sensitivity, and reserved words. Sloppy basics cause bugs later. Be precise.
