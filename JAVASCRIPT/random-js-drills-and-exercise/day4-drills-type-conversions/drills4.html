JavaScript Type Conversion Exercises

Exercise 1: Implicit Type Conversion (Coercion)

Predict the output and explain the result of expressions mixing different types, like:

  • Number + String
  • String - Number
  • Boolean + Number
  • String * String
  • Number / String

Exercise 2: Explicit Type Conversion

Convert the following values to the target type and log them:

  • Strings to numbers
  • Numbers to strings
  • Different values to boolean

Exercise 3: Understanding NaN

Predict when operations produce NaN and why:

  • Converting non-numeric strings to numbers
  • Invalid arithmetic operations
  • 0 / 0

Exercise 4: Truthy and Falsy Values

Identify whether the following values are truthy or falsy:

  • 0
  • "0"
  • ""
  • "false"
  • {}
  • []
  • NaN
  • ' '

Exercise 5: Type Conversion in Comparison

Predict outputs of comparisons involving coercion:

  • '5' == 5
  • '5' === 5
  • '5' == true
  • '0' == false
  • null == undefined
  • null === undefined

Exercise 6: Date Conversions

Convert Date objects to numbers and strings:

  • Number(date) → milliseconds
  • String(date) → human-readable string

Exercise 7: Handling Invalid Conversions

Predict outputs when converting invalid strings or tricky values to numbers:

  • '123abc'
  • '' (empty string)
  • ' ' (whitespace string)
  • 'true' and 'false'

Exercise 8: parseInt and parseFloat

Observe how parsing works with various strings:

  • Numeric strings with decimals
  • Strings with letters mixed in
  • Strings starting with letters

Exercise 9: Converting Objects to Primitives

Try converting objects to numbers or strings using:

  • valueOf()
  • toString()

Exercise 10: Null & Undefined Conversions

Predict the output when converting null and undefined to:

  • Number
  • String
  • Boolean
