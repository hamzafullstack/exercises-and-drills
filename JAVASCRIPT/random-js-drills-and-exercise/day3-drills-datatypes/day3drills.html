JavaScript Data Types | Exercise Sheet

A. Number

  • Create a variable with a number and log its typeof.
  • Divide a number by zero and see the result.
  • Divide a string by a number and check the output.
  • Compare NaN with NaN.

B. BigInt

  • Declare a BigInt number and log it.
  • Add 10n to your BigInt variable.
  • Try to add a BigInt with a normal number — see what happens.

C. Strings

  • Create strings using single, double, and backtick quotes.
  • Use template literals to insert a variable inside a string.
  • Print a string with a newline character (\n).

D. Boolean

  • Create a boolean variable and check its type.
  • Compare two numbers and log the result.
  • Compare 0 == false and 0 === false.

E. Null & Undefined

  • Declare a variable without assigning a value and log it.
  • Assign null to a variable and check its typeof.
  • Check null == undefined and null === undefined.

F. Symbol

  • Create two Symbols with the same description — compare them.
  • Check the typeof a Symbol.

G. Object

  • Create an object with name and age properties.
  • Access one property using dot notation.
  • Check typeof your object.

H. typeof Operator

  • Check typeof for: number, bigint, string, boolean, undefined, null, symbol, and object.
